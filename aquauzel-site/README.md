# AquaUzel — сайт-визитка

Одностраничный сайт-визитка компании **AquaUzel** (трубы, фитинги, тёплый пол, канализация и изоляция) с интерактивным прайс-листом и плавающими кнопками связи.

Чистый статичный сайт — **без сборки и зависимостей**. Можно открыть `index.html` напрямую или выложить на любой хостинг (GitHub Pages, Netlify, Vercel, обычный nginx/Apache).

## Возможности

- **Премиальный тёмный дизайн** в «водной» стилистике бренда (стекло, градиенты, анимации), шрифт Inter.
- **Интерактивный прайс-лист** — 8 категорий, 442 позиции из вашего Excel:
  - поиск по всем товарам в реальном времени с подсветкой;
  - вкладки по категориям, аккуратные таблицы, цены в тенге (₸);
  - корректно отображаются и матричные таблицы (каучуковая/рулонная изоляция).
- **Плавающие кнопки связи**, закреплённые в углах экрана при любой прокрутке:
  - справа — WhatsApp и звонок (трубка, `tel:`);
  - слева — Instagram.
- Полностью **адаптивно** (десктоп / планшет / телефон), с учётом `safe-area` на смартфонах.
- Доступность: контраст, фокус, `prefers-reduced-motion`, размеры тапов ≥ 44px.

## Структура

```
aquauzel-site/
├── index.html                  # страница
├── assets/
│   ├── css/styles.css          # стили (тема, дизайн)
│   ├── js/app.js               # каталог, прайс, поиск, меню
│   ├── js/prices-data.js       # данные прайса (window.AQUA_PRICES) — генерируется
│   ├── data/prices.json        # те же данные в JSON — генерируется
│   └── img/                     # логотип и иллюстрации (SVG)
│       ├── logo.svg  manifold.svg  favicon.svg
└── build/
    ├── aquauzel-price.xlsx     # исходный прайс (источник правды)
    └── parse_prices.py         # конвертер Excel → JSON/JS
```

## Как обновить прайс

1. Замените `build/aquauzel-price.xlsx` новой версией (сохранив структуру листов).
2. Выполните:
   ```bash
   cd aquauzel-site
   pip install openpyxl        # один раз
   python3 build/parse_prices.py
   ```
3. Обновятся `assets/data/prices.json` и `assets/js/prices-data.js` — сайт сразу подхватит новые цены.

Названия категорий, подписи и иконки настраиваются в словаре `SHEET_META` внутри `build/parse_prices.py`.

сохранив высоту ~40px в шапке).
